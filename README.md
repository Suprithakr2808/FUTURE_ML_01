# Support Ticket Classification System

## Overview
This project uses Machine Learning and NLP techniques to classify customer support tickets and assign priority levels.

## Features
- Text preprocessing using NLTK
- Ticket classification using TF-IDF + Logistic Regression
- Priority assignment logic
- Interactive command-line interface

## Tech Stack
- Python
- Scikit-learn
- NLTK

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python -m streamlit run ui.py
